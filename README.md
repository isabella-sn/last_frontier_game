# 🌌 Last Frontier — Jogo de Sobrevivência em Java

**Last Frontier** é um jogo de sobrevivência desenvolvido em **Java**, focado na aplicação prática dos conceitos de **Lógica de Programação** e **Programação Orientada a Objetos (POO)**.

O jogador enfrenta desafios em um ambiente hostil, tomando decisões estratégicas para **manter-se vivo**, gerenciar recursos e avançar no jogo.

---

## 🎯 Objetivo do Projeto

O projeto tem como objetivo:
- Aplicar conceitos fundamentais de **POO em Java**
- Trabalhar lógica de jogo e controle de fluxo
- Praticar modelagem de classes, métodos e estados
- Desenvolver um projeto completo para fins **acadêmicos**

Este jogo foi desenvolvido como parte de estudos em **LPOO (Linguagem de Programação Orientada a Objetos)**.

---

## 🕹️ Descrição do Jogo

Em **Last Frontier**, o jogador:
- Interage com um cenário de sobrevivência
- Enfrenta desafios progressivos
- Gerencia recursos disponíveis
- Toma decisões que impactam diretamente o andamento do jogo

O foco está na **lógica interna do jogo**, e não em gráficos avançados.

---

## 🧩 Conceitos Trabalhados

✅ Programação Orientada a Objetos

✅ Controle de fluxo
